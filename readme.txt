== Description ==

**Webillo Admin Customizer** is a minimalist plugin designed to improve brand consistency for WordPress websites. It automatically replaces the default WordPress login logo with your Site Icon (Favicon) and adds a clean welcome message to the dashboard.

Ideal for agencies and professionals who want to deliver a tailored experience to their clients without bloating the system with heavy code.

Key features:
* **Automatic Login Logo:** Uses your site's identity for the login screen.
* **Custom Dashboard Widget:** A professional touch for your admin area.
* **Ultra Lightweight:** Zero impact on site performance.

Developed with a "pixel-perfect" philosophy by [Webillo](https://webillo.com).

== Installation ==

1. Upload the `webillo-admin-customizer` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Your login page will automatically show your site icon.
